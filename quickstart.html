Brand Health KPI Dashboard - Quick Start Guide

Note: This guide will help you get started with the Brand Health KPI Dashboard and the Validation Tool.

Getting Started

Step 1: Open the Dashboard

Open the index.html file in a web browser to view the Brand Health KPI Dashboard.

The dashboard displays key performance indicators (KPIs) for brand health across Tier 1 markets, including metrics for Awareness, Familiarity, Consideration, and Intent.

Step 2: Access the Validation Tool

Click on the "Validation Tool" link in the header of the dashboard to access the validation tool.

The validation tool will automatically check for discrepancies between the CSV data and the hardcoded values in the code.

Step 3: Review Validation Results

The validation tool will display any discrepancies found between the CSV data and the code.

Currently, there are discrepancies in the Intent values for the US and China markets:

  • US Intent: CSV Value: 11.2%, Code Value: 24.2%
  • China Intent: CSV Value: 9.2%, Code Value: 18.5%

Step 4: Fix Discrepancies

Click the "Fix Intent Values" button to update the code with the correct values from the CSV file.

This will generate an updated version of the populateMetricCards.js file that you can download.

Replace the existing file with the downloaded file to fix the discrepancies.

Step 5: Refresh the Dashboard

After replacing the file, refresh the dashboard to see the updated values.

You can verify that the discrepancies have been fixed by checking the validation tool again.

Warning: The dashboard currently uses hardcoded data instead of directly parsing the CSV file. This can lead to discrepancies if the CSV file is updated but the code is not.

Consider updating the code to directly parse the CSV file to prevent future discrepancies.
